# 🚢 Titanic Survival Prediction Using Machine Learning

## 1. 🎯 Objective
Predict passenger survival on the Titanic based on features such as age, gender, passenger class, and fare.

## 2. 🛠 Tools & Technologies
- Python (Pandas, Scikit-learn, Matplotlib, Seaborn)  
- Excel  

## 3. 📊 Output / Result
- Model predictions for passenger survival.  
- Visualizations of feature importance (age, gender, class).  
- Performance metrics: Accuracy, Precision, Recall, F1-score.  

## 📁 Project Structure
/data → Raw & cleaned datasets  
/notebooks → Python scripts  
/images → Charts & visualizations
